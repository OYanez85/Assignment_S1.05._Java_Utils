Commands to compile and run the exercises:

Exercise 1 (ListDirectory.java):
    Compile: javac ListDirectory.java
    Run: java ListDirectory <directory_path>

Exercise 2 (DirectoryTree.java):
    Compile: javac DirectoryTree.java
    Run: java DirectoryTree <directory_path>

Exercise 3 (DirectoryTreeToFile.java):
    Compile: javac DirectoryTreeToFile.java
    Run: java DirectoryTreeToFile <directory_path> <output_file_path>

Exercise 4 (ReadTxtFile.java):
    Compile: javac ReadTxtFile.java
    Run: java ReadTxtFile <txt_file_path>

Exercise 5 (SerializeObject.java):
    Compile: javac SerializeObject.java
    Run to Serialize: java SerializeObject <output_file_path> serialize
    Run to Deserialize: java SerializeObject <output_file_path> deserialize
